# Privacidad

Jota AI Launcher es una aplicación local. No incluye servidor propio, analítica, publicidad ni telemetría de uso.

## Información que lee

- La presencia y versión de Codex, Claude Code, OpenCode, Node.js y la terminal disponible.
- El estado general de autenticación que publica cada CLI.
- Los nombres y versiones de plugins, skills y servidores MCP instalados.
- La carpeta de proyecto elegida por el usuario.

## Información que no recopila

- Contraseñas.
- Claves API.
- Tokens de sesión.
- Contenido de conversaciones.
- Código de los proyectos.
- Direcciones de correo de las cuentas conectadas.

El launcher no transmite este inventario a Jota ni a un servidor propio. Cuando el usuario inicia sesión, instala o actualiza una herramienta, se abre la terminal del sistema y la comunicación ocurre directamente entre esa herramienta y su proveedor.

Las preferencias del launcher se guardan en el perfil local del usuario del sistema. Las credenciales siguen siendo responsabilidad de Codex, Claude Code u OpenCode y no se copian dentro del instalador.
